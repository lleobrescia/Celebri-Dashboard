PersonalizarController source

  • Isolated Scope

Description

Edita o convite escolhido.
!! Não eh possivel editar ou trocar de modelo apos o envio !! Isso prejudicaria o APP
Pasta de origem : app/convite/personalizar
State : personalizar
Controller As : convite
Template Url : app/convite/personalizar/personalizar.html

Usa o serviço(s) do(s) servidor:

  • RetornarConvidadosConfirmados http://52.91.166.105/celebri/ServiceCasamento.svc/help/operations/RetornarConvidadosConfirmados
  • RetornarFormatacaoConvite http://52.91.166.105/celebri/ServiceCasamento.svc/help/operations/RetornarFormatacaoConvite
  • FormatacaoConvite http://52.91.166.105/celebri/ServiceCasamento.svc/help/operations/FormatacaoConvite
