CasalController source

  • Isolated Scope

Description

Receber e enviar dados do casal ( nome do casal, data de casamento e foto ).
Pasta de origem : app/casal
State : casal
Controller As : casal
Template Url : app/casal/casal.html

Usa o serviço(s) do(s) servidor:

  • AtualizarDadosCadastroNoivos http://52.91.166.105/celebri/ServiceCasamento.svc/help/operations/AtualizarDadosCadastroNoivos
